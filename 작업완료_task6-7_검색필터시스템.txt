3. **API 라우트**

   - `SearchResults` - 검색 결과 표시
   - `/resources` 페이지 - 통합 검색 페이지

## 주요 기능

### 검색 기능
- **전문 검색**: PostgreSQL의 고급 텍스트 검색 활용
- **실시간 자동완성**: 입력 중 추천 검색어 표시
- **다중 필터**: 타입, 레벨, 카테고리, 태그 조합 가능
- **정렬 옵션**: 최신순, 인기순, 투표순

### 카테고리 시스템
- **계층 구조**: 부모-자식 관계 지원
- **트리 뷰**: 시각적 계층 표시
- **자동 슬러그 생성**: URL 친화적 식별자
- **리소스 카운트**: 각 카테고리별 리소스 수 자동 집계

### 태그 시스템
- **동적 생성**: 사용자가 새 태그 즉시 생성 가능
- **인기 태그**: 사용 빈도순 정렬
- **자동완성 검색**: 빠른 태그 찾기
- **다중 선택**: 리소스당 여러 태그 지정 가능
